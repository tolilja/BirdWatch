Prerequisites
* Go to http://visualstudio.com/downloads
* Download and install Visual Studio 2017 Community including:
  * ASP.NET and web development
  * GitHub extensions for Visual Studio (Individual components)

BirdWatch project
* Go to Team Explorer and conncet to GitHub
* Clone https://github.com/tolilja/BirdWatch
* After cloning is completed, click on BirdWatch solution
* Rebuild solution

BirdWatch execution
* Client-Server
  * Select BirdWatch.html and Start Without Debugging
  * For more details see Palvelukuvaus.pdf
* Server
  * Select BirdWatchService.asmx and Start Without Debugging
